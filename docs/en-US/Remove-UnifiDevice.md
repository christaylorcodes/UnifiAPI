# Remove-UnifiDevice

## SYNOPSIS
Removes a device from a site.

## SYNTAX

```
Remove-UnifiDevice [-Device] <PSObject> [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Deletes a device from the UniFi Controller.
The device must be disconnected.

## EXAMPLES

### EXAMPLE 1
```
$device = Get-UnifiDisconnectedWaps | Where-Object { $_.name -eq 'Old AP' }
Remove-UnifiDevice -Device $device
```
