# Get-UnifiRogue

## SYNOPSIS
Retrieves rogue devices for a site.

## SYNTAX

```
Get-UnifiRogue [-SiteName] <String> [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
Returns devices detected as rogues on the specified site.
