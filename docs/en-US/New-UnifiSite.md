# New-UnifiSite

## SYNOPSIS
Creates a new site on the UniFi Controller.

## SYNTAX

```
New-UnifiSite [-Description] <String> [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Creates a new site with the specified description (display name).
The internal site name is auto-generated by the controller.

## EXAMPLES

### EXAMPLE 1
```
New-UnifiSite -Description 'Client ABC - Main Office'
```

### EXAMPLE 2
```
'Site A', 'Site B', 'Site C' | ForEach-Object { New-UnifiSite -Description $_ }
```
